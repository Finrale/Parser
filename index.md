# DataPulse — SkillCompass

Учебный семестровый проект: аналитика **IT-вакансий для аналитиков** — от сбора данных до ML-прогноза зарплаты и Streamlit-дашборда.

## Продукт

**SkillCompass** — компас навыков и зарплат для IT-аналитика:

- вакансии только по **четырём ролям** (см. ниже);
- какие навыки чаще требуют (SQL, BPMN, Jira, Python…);
- сравнение зарплат по ролям и городам;
- ML-прогноз зарплаты под ваш профиль;
- REST API и интерактивный дашборд.

### Целевые роли (узкая ниша)

| Ключ | Роль | Поисковый запрос |
|------|------|------------------|
| `business_analyst` | Бизнес-аналитик | бизнес-аналитик |
| `systems_analyst` | Системный аналитик | системный аналитик |
| `product_analyst` | Продуктовый аналитик | продуктовый аналитик |
| `data_analyst` | Аналитик данных | аналитик данных |

Сбор идёт **отдельным запросом на каждую роль**, дубликаты между выдачами удаляются.

## ML-задача (основная)

**Регрессия:** предсказать зарплату IT-аналитика по:

- типу роли (BA / SA / PA / DA);
- опыту, городу, формату работы;
- навыкам из описания (TF-IDF);
- источнику (HH / SuperJob).

**Дополнительно:** классификация роли по тексту вакансии; gap-анализ «каких навыков не хватает».

## Страницы дашборда

| Страница | Назначение |
|----------|------------|
| Overview | KPI по 4 ролям, топ навыков, зарплаты |
| Predictions | Калькулятор: «я SA с SQL и Jira — сколько?» |
| Data | Таблица вакансий, фильтр по роли |
| Monitoring | Статус HH / SuperJob |
